# React Framework

## Notes about react
- Created by Facebook
- Most popular based on a survey in Stackoverflow in 2021
- Based on NodeJS
- For a good tutorial check [here](https://www.w3schools.com/REACT)

## Basic steps
```bash
# create app
npx create-react-app my-react-app
# run the app
cd my-react-app
npm start
# now you see your app at localhost:3000
```
